### Description of Intended Activities, Products and Services

**Primary Business Activity:**

Hektor is a high-performance, open-source vector database company specializing in AI-powered similarity search and retrieval systems. The company develops and commercializes advanced vector search technology that enables organizations to build intelligent applications including semantic search, recommendation engines, retrieval-augmented generation (RAG) systems, and multimodal AI applications.

**Core Technology Product:**

Hektor Vector Database is a C++-based, SIMD-optimized vector database that delivers industry-leading performance for storing and querying high-dimensional vectors. The database achieves sub-3-millisecond query latency (p99) at 1 million vectors and scales to 1 billion+ vectors with distributed architecture. The product is differentiated by:

1. **Performance Leadership:** 50-70% faster than major competitors (Pinecone, Weaviate, Milvus) through advanced SIMD optimization using AVX2 and AVX-512 instructions
2. **Industry-First Innovation:** Perceptual quantization technology based on SMPTE ST 2084 (PQ curve) standard, reducing memory footprint by 65% while maintaining 98.1% recall accuracy
3. **Cost Efficiency:** 60-80% lower total cost of ownership compared to managed cloud alternatives
4. **Open Source Foundation:** MIT-licensed core product enabling community adoption and ecosystem development

**Products and Services Portfolio:**

1. **Open Source Core (Free):**
   - Full-featured vector database with HNSW indexing
   - Self-hosted deployment on any infrastructure
   - Community support through GitHub and Discord
   - MIT License with no commercial restrictions

2. **Enterprise Support Subscriptions ($5,000 - $500,000/year):**
   - Basic Support: Email support, 48-hour SLA, $5,000/year
   - Standard Support: 24-hour SLA, monthly check-ins, $15,000/year
   - Premium Support: 24/7 availability, dedicated engineer, $50,000/year
   - Enterprise Support: Custom SLAs, technical account manager, architecture review, $100,000-$500,000/year

3. **Professional Services ($200/hour, $5,000/day consulting):**
   - Migration services from existing vector databases ($25,000-$100,000 per project)
   - Custom feature development and optimization
   - Training programs for engineering teams
   - Architecture consulting and performance tuning
   - Integration assistance with ML frameworks

4. **Managed Service - Hektor Cloud (Launch: 2027):**
   - Fully managed vector database service
   - Multi-cloud deployment (AWS, Azure, GCP)
   - Automatic scaling and high availability
   - 40-50% cost advantage over Pinecone
   - Usage-based pricing model
   - Enterprise SLAs: 99.9-99.99% uptime

5. **OEM and Partnership Revenue (2027+):**
   - Cloud marketplace listings (AWS, Azure, GCP) with 20% revenue share
   - Technology partnerships with ML platforms
   - Integration partnerships with LangChain, LlamaIndex, Hugging Face

**Target Use Cases:**

- **Semantic Search:** Natural language understanding and document retrieval
- **RAG Systems:** Retrieval-augmented generation for large language models
- **Recommendation Engines:** Real-time personalization and content discovery
- **Image/Video Search:** Visual similarity and content-based retrieval
- **Fraud Detection:** Anomaly detection and pattern recognition
- **Customer Service:** Intelligent chatbots and support automation

The business model leverages the "open source, commercial support" strategy pioneered by successful infrastructure companies like Redis Labs, MongoDB, and Confluent. This approach drives rapid adoption through free open-source availability while monetizing enterprise support, professional services, and managed cloud offerings.
