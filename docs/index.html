Hybrid storage: SQLite for caching + PostgreSQL pgvector for persistence. Automatic fallback, write-through caching, unified interface.

/// Quick Start

Get Started

# Clone repository
git clone https://github.com/amuzetnoM/vector_studio.git
cd vector_studio

# Build
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build

# Python usage
import pyvdb

db = pyvdb.VectorDatabase("./vectors", 512)
db.add_text("High-performance vector search")
results = db.search("fast similarity search", k=10)
