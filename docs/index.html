Features

Enterprise-Grade Capabilities

Built for performance, scalability, and production workloads

SIMD Optimization

AVX2/AVX-512 accelerated distance computations for blazing-fast similarity search on modern CPUs.

HNSW Indexing

Hierarchical Navigable Small World graph for O(log n) approximate nearest neighbor search with high recall.

Local Embeddings

ONNX Runtime inference for text and images without API calls. Complete privacy and no latency overhead.

Cross-Modal Search

Unified 512-dimensional space enables searching images with text queries and vice versa.

Memory-Mapped Storage

Zero-copy vector access via mmap for efficient I/O and datasets larger than RAM.

Python Bindings

Native Python API via pybind11 for seamless integration with data science workflows.

Comprehensive Logging

Thread-safe multi-level logging with anomaly detection. Zero blind spots - all operations monitored.

Universal Data Ingestion

Full read/write support for XML, PDF, SQLite, Parquet, CSV, JSON, Excel, and PostgreSQL pgvector.

Quick Start

Get Running in Minutes

Automated setup for Windows, macOS, and Linux

# Clone repository
git clone https://github.com/amuzetnoM/vector_studio.git
cd vector_studio

# Build
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build

# Python usage
import pyvdb

# Create database
db = pyvdb.create_database("./vectors")

# Add text with metadata
db.add_text("High-performance vector search", {
    "type": "document",
    "date": "2025-01-04"
})

# Search
results = db.search("fast similarity search", k=10)
for r in results:
    print(f"{r.score:.4f}: {r.text}")
