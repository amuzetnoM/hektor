# v4.0.0

**Release Date:** January 21, 2026  
**Status:** Major Release

## Overview

Hektor v4.0.0 is a major release that adds significant new features and technology enhancements since v3.0.0. This release includes advanced perceptual quantization research, comprehensive competitor analysis, improved build systems, and various performance optimizations.

## New Features

### Perceptual Quantization Research
- **Advanced Quantization Techniques**: Implementation of perceptual quantization for HDR/image embeddings
- **Research Papers**: Three comprehensive research papers on quantization evolution
  - Neural Latent Quantization
  - Temporal Redundancy Beyond Motion Vectors
  - Quantum Quantization
- **Perceptual Manifolds**: Detailed documentation on quantization evolution
- **Performance Improvements**: Up to 4x speedups with optimized quantization methods

### Competitor Analysis & Research
- **Comprehensive Vector Database Analysis**: Deep dive into competitor landscape
- **Interactive Visualization Dashboard**: HTML-based charts and comparisons
- **Billion-Scale Metrics**: Performance benchmarks at scale
- **Stakeholder Documentation**: Complete competitive positioning analysis

### Build & Installation Improvements
- **Automated Build Script**: Comprehensive build automation
- **Improved pip Installation**: Enhanced Python package installation
- **Architecture Detection**: Automatic AVX flags detection
- **CI/CD Optimizations**: New optimized CI workflows

### Documentation Updates
- **Performance Tuning**: Comprehensive benchmarks and optimization guides
- **Competitor Analysis**: Detailed comparison documents
- **Configuration Management**: Extracted config constants for better maintainability
- **Research Integration**: Integration of research findings into documentation

## Security Updates
- **Dependency Update**: Upgraded `tar` package from 7.5.2 to 7.5.3
  - Fixes vulnerability related to absolute linkpath sanitization

## Technical Details
- **Major Version Bump**: 3.0.0 → 4.0.0 (significant new features and enhancements)

## Upgrading

### Docker
```bash
docker pull ghcr.io/amuzetnom/hektor:v4.0.0
```

### From Source
```bash
git checkout v4.0.0
cmake -B build -G Ninja
cmake --build build
cd build && pip install .
```

### Python Package
```bash
pip install --upgrade hektor-vdb
```
