## Quick Start

### For Marketing Managers
1. Start with `strategy/90-day-plan.md` - Your execution roadmap
2. Review `workflow/team-roles.md` - Assign responsibilities
3. Set up tracking via `analytics/tracking-setup.md`
4. Begin Day 1 using `social-media/content-calendar.md`

### For Content Creators
1. Review `copy/value-propositions.md` - Core messaging
2. Use templates in `templates/` for consistent output
3. Pull social posts from `social-media/` directory
4. Follow `workflow/publishing-checklist.md` before posting

### For Designers
1. Assets in `assets/svgs/` - All vector graphics
2. Desktop app assets in `assets/desktop-app/`
3. Maintain brand consistency per positioning guidelines
